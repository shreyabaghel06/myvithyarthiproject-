# LIBRARY MANAGEMENT SYSTEM 


OVERVIEW


This Python Library Management System is designed to help libraries efficiently track their books and users. It supports essential operations like adding new books, viewing inventory, issuing, and returning books, making it a practical solution for schools and small libraries. The project emphasizes modular, maintainable code and user-friendly interactions through a graphical interface.


FEATURES


1.Add new books to the library database

2.View and search books in inventory

3.Issue books to registered users

4.Return books and update availabilitY


TECHNOLOGIES/TOOLS USED


1.Python 3.13

2.VS Code or other editor for development


INSTALLATION & RUNNING STEPS 


1.Ensure Python 3.11 is installed.

2.Run the main application:

python main.py
